# USEDヨーヨーマーケット🪀
【URL】https://heroku-deploy-usedyoyo-app.herokuapp.com
# 概要
ヨーヨープレイヤー同士でヨーヨーを取引できるアプリケーションです。<br>ヨーヨーを整理して、欲しかったヨーヨーをゲットして、お互いHAPPYになろう!

# 開発した経緯
プログラミングを勉強する中で、オリジナルのWEBサービスを開発したいという思いがあり、私の趣味である「競技ヨーヨー」に関するアプリを開発することにしました。<br>
使わなくなった・使えなくなった中古ヨーヨー「USEDヨーヨー」の価値をもう一度見直し、本当に必要とする人のもとへ行き届いて欲しいという思いでこのアプリを開発しました。<br>ヨーヨーを売りたい人とヨーヨーを探している人同士の取引を実現し、双方が満足できるヨーヨートレードの場を提供することで、中古ヨーヨーの価値を循環させることを目的としている。<br>アプリのロゴには、ヨーヨーが回転するように、ヨーヨーの価値も循環させたいという思いが込められています。
